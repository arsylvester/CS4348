Files Included:
	CPU.java - The parent process and simulates the CPU of a computer system.
	Memory.java - The child process and simulates the Memory of a computer system.
	sample5.txt - This input file prints the Fibonacci Sequence from n = 0 to n = 14. Also it shows that interrupts don't affect user code.
	Project 1 Summary - The summary of the project.

Compile and run instructions:
	1. Put CPU.java, Memory.java, and the input files in one directory.
	2. Compile CPU.java and Memory.java
		javac CPU.java
		javac Memory.java
	3. Run CPU using inputfile name and timer duration as parameters.
		java CPU sample5.txt 25
